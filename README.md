# Task 1: Responsive Landing Page

## Objective:
To build a clean and responsive landing page using HTML and CSS.

## Tools Used:
- Visual Studio Code
- Chrome Browser
- Live Server Extension

## Features:
- Header with logo and navigation links
- Hero section with heading, paragraph, and CTA button
- Footer with social links
- Fully responsive using Flexbox and Media Queries

## Outcome:
Learned HTML structure, CSS layout, Flexbox/Grid, and media queries.
